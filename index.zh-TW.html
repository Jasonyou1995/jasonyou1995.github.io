業務範圍

落地級 AI，而非概念驗證

大多數 AI 專案死在 Demo 和上線之間。我專注於彌合這一鴻溝：交付能夠承擔真實業務任務的 Agent 系統，並在離開後讓你的團隊能夠獨立維運。

01

Agent 設計與編排

基於 LangGraph 與 n8n 的多步驟智能體工作流，聚焦一個真實業務流程，在關鍵環節設置人機協作檢查點。

02

生產工程化

基於 Cloud Run 的 FastAPI 服務，具備完善的錯誤處理、可觀測性與告警機制。元件可稽核，拒絕黑箱。

03

可信與安全架構

將研究中的提案者 / 挑戰者 / 裁決者審查模式落地應用，確保 Agent 輸出在執行前經過校驗，而不是盲目執行。

04

移交與賦能

交付完整原始碼、維運手冊與培訓 session，確保你的團隊真正掌握系統。無鎖定，無依賴。

研究

多智能體可信性與安全

我的博士研究處於自主智能體、共識機制設計與安全的交叉領域。工程實踐與學術研究相互驅動。

校準型多智能體決策系統

如何信任自主智能體的輸出？我的研究開發了提案者 / 挑戰者 / 裁決者架構，獨立智能體在決策執行前進行論證與裁決，作為實現可校準、可辯護自動化的路徑。我在學術中研究的同一架構，正是我在客戶系統中部署的安全層：輸出經過挑戰與裁決，而非盲目執行。

主題：多智能體系統、共識與信任、校準預測、區塊鏈與去中心化系統安全。
